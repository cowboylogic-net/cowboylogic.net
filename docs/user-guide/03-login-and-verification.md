# 03 Login and Verification

This section covers confirmed login methods:

- Email + password
- Google login
- Email code verification flow (when required)

## Where to find Login/Logout

- **Desktop:** `Login` / `Logout` is in the **header**.
- **Mobile:** `Login` / `Logout` is in the **burger menu**.

## Task: Log in with email and password

1. Open `<SITE_URL>`.
2. Click `Login`.
3. Enter your email and password.
4. Click the login/continue button.

What you should see:

- You are logged in and returned to the site.
- Account links become available (for example: `My Profile`, `My Orders`, `Logout`).

If it doesn’t work:

- Check email/password spelling.
- If you are asked for a code, follow the next task.

## Task: Complete email code verification (when requested)

This can happen during login (for example, if verification is required).

1. After you submit login, wait for the code email.
2. Enter the code.
3. Submit verification.
4. If available, use `Resend code` (it may be disabled during a short cooldown).

What you should see:

- Successful verification and login.

If it doesn’t work:

- Wait 1–2 minutes and check Spam/Junk.
- Use `Resend code` when the cooldown ends.
- Contact `<SUPPORT_EMAIL>` if codes never arrive.

## Task: Log in with Google

1. Open the `Login` page.
2. Click the Google sign-in button.
3. Choose your Google account.
4. Confirm sign-in.

What you should see:

- You are logged in and returned to the site.

If it doesn’t work:

- Try email + password login instead.
- If the site indicates your email is not verified, complete verification first.

## Task: Log out

1. Click `Logout` (desktop header or mobile burger menu).
2. Confirm you are back in guest mode.

What you should see:

- Private pages require login again.

If it doesn’t work:

- Refresh the page once.
- Close and reopen the browser tab, then check again.
