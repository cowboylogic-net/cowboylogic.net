## Common terms

- **Home**: Main start page of the website.
- **CLStrategies**: Section with strategy-related content pages (dropdown in navigation).
- **CLPublishing**: Section with publishing-related content pages (dropdown in navigation).
- **CLP Book Store / Bookstore**: Main store for regular book shopping.
- **Partner Store**: Store view with partner pricing and partner purchase rules (requires partner-enabled access).
- **Cart**: Your current shopping cart (items and quantities you plan to purchase).
- **Checkout / Checkout with Square**: Payment step opened from `Cart`. (Label may appear as `Checkout` or `Checkout with Square`.)
- **My Profile / Profile**: Your account details page (personal info, avatar, password).
- **My Orders / Orders**: Your order history page (order list, totals, statuses).
- **Favorites**: Books you saved for quick access. Requires being logged in.
- **Admin Dashboard**: Admin-only control area with links to admin tools.
- **User role**: Standard customer access (shop, cart, orders, profile).
- **Partner role**: Access to `Partner Store` and partner pricing rules (may require minimum quantities).
- **Admin role**: Access to admin tools (book management, user management, newsletter, admin order actions).
- **Super Admin**: Highest admin role. Usually marked with a crown icon (`ðŸ‘‘`). This account cannot be deleted (and may be restricted from role changes).
- **Verification code**: Email code used to verify/login when required.
- **Newsletter subscription**: A profile setting that controls whether you receive email updates.
- **Edit**: Button used to enable editing for a specific field or section.
- **Save**: Confirm changes (for example profile/password updates).
- **Cancel**: Discard changes and close the edit mode.

## Admin content editing terms

- **Edit Page**: Enable editing mode on pages that support admin content editing.
- **Save Draft**: Save admin page changes without publishing to all users.
- **Publish**: Make admin page changes visible to users.
